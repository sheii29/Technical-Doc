Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

It is widely used in web development, data science, automation, and software development.

The Python Software Foundation manages its development and releases new versions regularly.

print("Hello, World!")
  • High-level language
  • Interpreted
  • Multi-paradigm
Data Types

Python has a variety of built-in data types, making it versatile for different applications.

The main data types are int, float, str, list, tuple, dict, and set.

Each data type serves different purposes and has unique properties.

Type checking in Python is dynamic, meaning types are checked during runtime.

x = 10 # int y = 3.14 # float
  • int
  • float
  • str
  • list
  • dict
Control Flow

Control flow structures in Python include if, elif, else, for, while, and try-except.

These structures allow the programmer to control the flow of the program based on conditions.

Control flow helps in building more complex logic by branching and looping.

Python also supports exception handling for error management.

if x > 0: print("Positive")
  • If-else
  • For loop
  • While loop
  • Exception handling
Functions

Functions in Python allow code to be reusable and organized.

A function is defined using the def keyword and can take parameters.

Python also supports lambda functions, which are anonymous functions used for short operations.

Functions can return values using the return keyword.

def greet(name): return f"Hello, {name}"
  • def keyword
  • Parameters
  • Lambda functions
  • Return statement
Modules and Packages

Python supports modular programming by allowing code to be organized in modules and packages.

A module is a single Python file, and a package is a directory of modules.

Modules can be imported into other programs using the import keyword.

This allows for better code organization and reusability.

import math from datetime import datetime
